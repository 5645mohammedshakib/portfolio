## 🚀 How to Run Locally

Since this portfolio is built as a highly responsive, static site using vanilla CSS and JavaScript, you can run it in two ways:

### Option 1: Direct Execution
1. Clone the repository:
   ```bash
   git clone https://github.com/5645mohammedshakib/portfolio.git
   ```
2. Double-click [index.html](index.html) to open it directly in any web browser.

### Option 2: Using Vite Development Server
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local server:
   ```bash
   npm run dev
   ```
3. Open the local address (e.g., `http://localhost:5173`) in your browser.
